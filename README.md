## 🍀屏幕水印ScreenWatermark 简介

- 使用Python + Qt开发的一款桌面水印工具。

- 支持多屏显示

- 支持开机自启动

- 支持富文本显示

- 支持全屏文字、图片水印

- 支持局部文字、图片水印
